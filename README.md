## Опис на проектот

PlacePin е мобилна Android апликација изработена како индивидуален проект по предметот „Развој на мобилни апликации“.

Апликацијата претставува бизнис директориум кој им овозможува на корисниците да пребаруваат и прегледуваат компании организирани по категории.

## Функционалности

- Додавање нови компании
- Чување податоци во SQLite база
- Пребарување на компании
- Филтрирање по категории
- Приказ на детали за компанија
- Приказ на лого на компанија
- Отворање локација во Google Maps
- Повикување на телефонски број
- Отворање на веб-страница
- Геолокациска функционалност
